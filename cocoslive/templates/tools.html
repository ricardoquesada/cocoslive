Client Tools

cocos Live client is included within the cocos2d for iPhone package. You can download the latest version here:

Download: http://code.google.com/p/cocos2d-iphone

For information on how to use, please read this post: cocos Live: a high score server for cocos2d. The client is coded in objective-c and can be used with cocos2d or with any other engine like SIO2 or Oolong.
